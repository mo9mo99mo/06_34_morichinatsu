◇ファイルについて
index.php    ---入力（投稿）フォーム
insert.php   ---投稿をDBに登録処理するページ
select.php   ---DBに登録したデータを一覧表示
edit.php     ---DBに登録したデータを更新変更するページ
update.php   ---edit.phpで変更した内容にDBを更新処理するページ
delete.php   ---DBに登録したデータを削除処理するページ
